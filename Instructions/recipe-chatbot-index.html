Recipe Chatbot Evaluation Exercise

Learn AI evaluation fundamentals through hands-on tasks

Scenario: Farm Business Recipe Chatbot

Context: You're helping evaluate an AI chatbot for Munson's Pickles and Preserves Farm that assists customers with recipe recommendations, meal planning, and cooking guidance using farm products.

Your Role: AI evaluation specialist responsible for ensuring the chatbot meets customer needs and business objectives.

Learning Objectives

  • Identify relevant user needs for AI systems
  • Evaluate response quality using systematic criteria
  • Select appropriate evaluation metrics for business contexts
  • Create comprehensive test sets for AI validation
  • Apply ACRUE framework for response assessment
1
Identify User Needs
Select the 3 most relevant user needs for a farm business recipe chatbot from a list of 9 options. Immediate feedback will confirm whether you picked correctly.
Start Task 1
2
Evaluate Response Quality
Compare pairs of AI responses to customer queries and select the higher quality response that better satisfies each customer's need.
Start Task 2
3
Select Evaluation Metrics
From 6 available built-in evaluation metrics, select the 3 most relevant for this use case. View a sample evaluation dashboard after submitting.
Start Task 3
4
Create a Smoke Test Set
Enter 9 diverse test prompts (3 per user intent) to serve as a smoke test for the chatbot. System will check coverage and provide feedback.
Start Task 4
5
Manual ACRUE Evaluation
Using the 9 test prompts, rate responses against ACRUE dimensions (Accurate, Complete, Relevant, Useful, Exceptional) on a 1-5 scale.
Start Task 5

Exercise Flow

Complete tasks in order for the best learning experience. Each task builds on knowledge from previous tasks and provides immediate feedback.

Total Duration: 25-30 minutes | Format: Interactive exercises with instant feedback
