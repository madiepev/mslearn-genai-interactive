# Quick Project Guidelines

A lightweight tool to create project guidelines in 5 minutes for solo developers.

## How to Use

1. Open **`quick-generator.md`**
2. Answer the 4 simple question groups
3. Copy the mini template and fill in your answers
4. Done! You have customized project guidelines

## What You Get

- Essential project structure and standards
- Tech stack documentation
- Simple quality checklist
- Definition of done criteria

Perfect for solo projects and small teams who want structure without complexity.
